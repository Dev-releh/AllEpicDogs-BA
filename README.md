All Epic Dogs – Mobile App Prototype (Business Analysis Project)
📌 Project Overview
A complete business analysis for a pet services mobile app prototype that connects pet owners with professional dog walking and care services. This was a self-initiated independent project – I identified the gap in pet service booking, defined the scope, and completed all analysis from start to finish.

📊 Key Highlights
📄 Functional and non-functional requirements defined

📝 28 Agile user stories written with acceptance criteria

🔍 6 critical gaps identified (CVV storage, missing appointment history, no automated reminders)

🖼️ 20+ prototype screenshots analyzed

📋 Complete Business Requirements Specification (BRS) delivered

🛠️ Tools Used

Figma – Prototype

Google Docs – Business Requirements Specification
